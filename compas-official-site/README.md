# Compas Pasta House 康帕斯義麵屋官方網站

這是一個靜態品牌官網專案，目標是讓 Google Search 與 AI Chat 更容易理解並搜尋到品牌。

## 頁面

- `index.html`：首頁與品牌介紹
- `menu.html`：菜單
- `reservation.html`：訂位展示表單
- `location.html`：Google Map 與店舖資訊
- `line.html`：官方 LINE 加入好友

## 正式網址

- Cloudflare：`https://compas-official-site.compaspastahouse.workers.dev/`
- 未來連接自訂網域後，需同步更新 canonical、Open Graph、JSON-LD、`robots.txt` 與 `sitemap.xml`。

## SEO/GEO 已先放入的內容

- 每頁都有清楚的 `<title>` 與 `meta description`
- 首頁有 Restaurant JSON-LD 結構化資料
- 品牌中英文名稱重複出現在重要位置
- 有 `robots.txt` 與 `sitemap.xml`
- 店舖資訊、Google Map、官方 LINE 分成獨立頁面
